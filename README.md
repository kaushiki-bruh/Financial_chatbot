# Financial_chatbot
A keyword-based financial chatbot built using Python and pandas, combined
with exploratory data analysis in Jupyter Notebook.

The project analyzes financial data for Microsoft, Tesla, and Apple
from 2022–2024 and allows users to query metrics such as revenue,
net income, and assets. The dataset was manually extracted from SEC
filings for accuracy and verified insights.
## Tech Stack
- Python
- Pandas
- Matplotlib (visualization)
- Jupyter Notebook
## Dataset
- Cleaned financial data for Microsoft, Tesla, and Apple.
- Fiscal years: 2022–2024
- Source: SEC filings
## Features
- Keyword-based question answering
- Supports multiple companies and years
- Cleaned and structured financial dataset
- Simple command-line interface
## Key Insights
- Microsoft shows consistent revenue growth
- Tesla displays higher volatility
- Apple shows stable, mature revenue patterns
